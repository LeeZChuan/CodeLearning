##   2013年西电计算机问题A

     题目描述：
     定义一个新的斐波那契数列；
     F(0)=7；
     F(l)=ll;
     F(n)=F (n-l) +F (n-2) ; (n>=2)
     输入：
     输入有多组；首先输入一个N (NOIOO)，代表要输入的测试用例的个数；接下 来输入N个数字ni(ni«100),数字间用空格隔开。
     输出：
     求F(n)能否被3整除，若能整除输出‘yes'，否则输出‘no'。
     样例输入：
     3
     0 1 2
     样例输出：
     no
     no
     yes
